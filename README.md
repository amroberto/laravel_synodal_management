# Synodal Management (Legacy - Laravel MVC)

> **Status:** Arquivado (Legacy)

Este repositório contém a **primeira versão** do projeto **Synodal Management**, desenvolvida utilizando a arquitetura tradicional do Laravel baseada em **Models, Views e Controllers (MVC)**.

O projeto foi criado com o objetivo de apoiar a gestão administrativa de comunidades, paróquias e lideranças de um sínodo da igreja, servindo como base para estudos e evolução da aplicação.

## Tecnologias

* PHP
* Laravel
* Blade
* Bootstrap
* AdminLTE
* MySQL

## Status do projeto

Este projeto não receberá novas funcionalidades.

O desenvolvimento continua em uma nova versão, construída do zero com uma arquitetura mais moderna utilizando:

* Laravel 13
* Livewire
* Flux UI
* Tailwind CSS
* Components
* Traits
* Services

Esta nova arquitetura proporciona maior reutilização de código, melhor organização e uma experiência de desenvolvimento mais moderna.

## Objetivo deste repositório

Este repositório permanece disponível para registrar a evolução do projeto e demonstrar a primeira implementação da aplicação utilizando a arquitetura MVC do Laravel.
